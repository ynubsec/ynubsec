# Security Policy

## Supported Versions

Only the latest version of my tools and research are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < latest| :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within my projects or research, please send an encrypted report to **ynubsec@gmail.com**. 

Please provide:
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact analysis.

I will acknowledge receipt of your report within 48 hours and provide a timeline for resolution if applicable. Responsible disclosure is appreciated.
